# Jupyter-like Features

AdeloopFlow provides several Jupyter-like features to enhance your coding experience. This document provides an overview of all available features.

## Feature Overview

1. [Rich Output Support](RICH_OUTPUT_SUPPORT.md) - Display DataFrames, plots, and other complex data structures in enhanced visual formats
2. [Magic Commands](MAGIC_COMMANDS.md) - Special commands prefixed with % that provide enhanced functionality
3. [Shell Command Support](SHELL_COMMAND_SUPPORT.md) - Execute system commands directly from code cells using the ! prefix
4. [Execution Count Tracking](EXECUTION_COUNT_TRACKING.md) - Sequential numbering system for code cell executions

## Getting Started

To use these features, simply use them as you would in a Jupyter notebook. Each feature is designed to work seamlessly with the others to provide a familiar and powerful coding environment.
